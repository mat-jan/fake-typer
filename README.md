# Fake Typer Pro ⌨️🚀

Prosty i efektowny symulator "hakerskiego" pisania, idealny do hostowania na **GitHub Pages**. Wklejasz dowolny tekst, a następnie uderzasz w przypadkowe klawisze, by na ekranie pojawiała się profesjonalna treść znak po znaku.

## ✨ Funkcje
- **Custom Text:** Wklejasz własny kod, artykuł lub dokument.
- **Manual Control:** Tekst pojawia się tylko wtedy, gdy naciskasz klawisze (to nie jest animacja, to Ty "piszesz").
- **Hacker Aesthetic:** Design inspirowany terminalem (zielony tekst na czarnym tle).
- **Auto-scroll:** Strona automatycznie przewija się w dół wraz z przybywaniem tekstu.
- **Escape Mode:** Naciśnij klawisz `ESC`, aby wrócić do edycji tekstu.

## 🚀 Jak uruchomić na GitHub Pages?

1. Stwórz nowe repozytorium na GitHub (np. `fake-typer`).
2. Prześlij plik `index.html` do głównego folderu repozytorium.
3. Przejdź do **Settings** -> **Pages**.
4. W sekcji **Branch** wybierz `main` (lub `master`) i kliknij **Save**.

## 🛠️ Jak używać?

1. Otwórz stronę w przeglądarce.
2. Wklej tekst, który chcesz "napisać" (np. kod źródłowy w C++ lub Pythonie).
3. Kliknij **"Zacznij symulację"**.
4. Uderzaj dłońmi w klawiaturę jak szalony – na ekranie będzie pojawiał się Twój tekst.
5. Aby zmienić tekst, naciśnij **ESC**.
